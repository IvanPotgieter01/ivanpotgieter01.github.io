01 · About

A bit about how I work

I'm a Software Engineer at the BMW IT Hub South Africa, focused on cloud, DevOps, security, and applied AI. I automate and secure infrastructure across AWS and Azure using Terraform, GitHub Actions, and a lot of PowerShell and Bash - with a strong DevSecOps focus: vulnerability management and code scanning, GitHub Advanced Security, TLS/PKI and certificate management, release management, security-posture reporting, and test automation.

I currently work on a serverless, AI-powered knowledge platform built around retrieval-augmented generation (RAG), and I'm just as hands-on with applied AI day to day - context engineering for large language models and building AI-assisted engineering workflows with coding agents and the Model Context Protocol (MCP).

I enjoy turning manual, risky processes into automated, secure, and repeatable ones, and I'm AWS Certified Solutions Architect - Associate.
